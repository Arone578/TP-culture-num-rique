# TP Culture Numérique – AlgoBox et GitHub

## Description
Projet réalisé en groupe dans le cadre d’un TP de culture numérique.
L’objectif de ce travail est de découvrir la programmation avec AlgoBox
et le travail collaboratif à l’aide de GitHub.

## Organisation du projet
Les programmes AlgoBox sont regroupés dans le dossier src/.

## Contenu
- src/test.algo.alg : programme de test
- src/menu_calcul.algo.alg : programme avec menu de calcul
- src/stats_nombre.alg : programme de statistiques (somme, maximum)
- src/mot_de_passe.alg : programme de vérification de mot de passe

## Exécution
1. Ouvrir AlgoBox
2. Ouvrir un fichier .alg depuis le dossier src
3. Cliquer sur Exécuter
